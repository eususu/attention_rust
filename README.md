# attention_rust
`attention server in rust`
This project made for testing.

## listen

- bind address `is 0.0.0.0`
- bind port is `8008`

## build and run

`cargo build`
`cargo run`

## end points

| end point | meaning |
| - | - |
| /service | manage service infomations |
| /qa{service_name} | manage question-answering informations about {service_name} |
